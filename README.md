# 🌌 Astro-Cycles Calculator
(A calculadora de ciclos y aspectos astronómicos)

## 🎯 Descripción
Astro-Cycles es una aplicación web ligera y autónoma diseñada para visualizar complejos tránsitos planetarios, ciclos sinódicos e índices armónicos. Se construye exclusivamente con tecnologías web
nativas (HTML, CSS, JavaScript), garantizando una experiencia de usuario sin dependencias externas ni instalaciones complejas.

---

## ✨ Características Principales
*   **🛰️ Tránsitos Planetarios (T-T):** Cálculo y visualización detallada de la separación angular entre pares de planetas a lo largo del tiempo.
*   **🌙 Tránsitos Natales (T-N):** Muestra la interacción entre los planetas actuales y una carta natal predefinida.
*   **⚖️ Índice Armónico:** Sistema de puntuación que representa la "tensión" o "armonía" general basada en los aspectos activos (Conjunción, Cuadratura, Trígono, etc.).
*   **📈 Gráfico Interactivo:** Una visualización dinámica que permite inspeccionar fechas, ángulos y ciclos específicos con alta precisión.
*   **⚙️ Totalmente Configurable:** Permite al usuario ajustar dinámicamente parámetros clave como el rango de fechas, el orbe de los aspectos y el suavizado del índice.
*   **🚀 Autónoma:** Funcionamiento garantizado en cualquier navegador moderno sin la necesidad de servidor backend o dependencias externas.

# 🌌 Astro-Cycles Calculator
(An Astronomical Cycle and Aspect Calculator)

## 🎯 Description
Astro-Cycles is a lightweight, standalone web application designed to visualize complex planetary transits, synodic cycles, and harmonic indices. Built using only native web technologies (HTML, CSS,
JavaScript), it guarantees a dependency-free user experience without needing external servers or complex installations.

---

## ✨ Core Features
*   **🛰️ Planetary Transits (T-T):** Calculates and visualizes the angular separation between pairs of planets over time (T-T transits).
*   **🌙 Natal Transits (T-N):** Displays the transits of current planetary placements against a predefined birth chart (T-N transits).
*   **⚖️ Harmonic Index:** A comprehensive scoring system representing the overall "tension" or "harmony" based on active aspects (Conjunction, Square, Trine, etc.).
*   **📈 Interactive Graph:** A dynamic visualization tool that allows users to inspect specific dates, angles, and cycles with high precision.
*   **⚙️ Highly Customizable:** Users can dynamically adjust key parameters such as the date range, the aspect orb, and the index smoothing factor.
*   **🚀 Standalone Operation:** Guaranteed to run on any modern web browser without the need for a backend server or external dependencies.

---

## ⚙️ Setup and Deployment

This project is designed to be a static website. It requires no backend or database services.

**1. Local Execution (Testing):**
   *   Simply open the `index.html` file in any modern web browser (Chrome, Firefox, etc.).

**2. Production Deployment:**
   *   The entire project can be deployed directly to any static hosting service (e.g., GitHub Pages, Netlify, Vercel). You only need to upload the contents of the root folder.
