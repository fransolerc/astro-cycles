# Astro-Cycles Calculator

## Descripción

Astro-Cycles es una calculadora de ciclos y aspectos astronómicos que visualiza tránsitos planetarios, ciclos sinódicos y un índice armónico. Es una aplicación web ligera y autónoma, construida únicamente con HTML, CSS y JavaScript, sin dependencias externas.

## Características

- **Cálculo de Tránsitos Planetarios**: Calcula y visualiza la separación angular entre pares de planetas a lo largo del tiempo (tránsitos T-T).
- **Tránsitos Natales**: Calcula y visualiza los tránsitos de planetas actuales contra una carta natal (tránsitos T-N).
- **Índice Armónico**: Muestra un índice de "tensión" o "armonía" basado en los aspectos activos (conjunción, cuadratura, trígono, etc.).
- **Gráfico Interactivo**: Visualiza los ciclos en un gráfico interactivo que permite inspeccionar fechas y ángulos específicos.
- **Configurable**: Permite ajustar el rango de fechas, el orbe de los aspectos y el suavizado del índice.
- **Sin Dependencias**: Funciona en cualquier navegador moderno sin necesidad de instalar nada.

## Despliegue

Este proyecto está diseñado para ser servido como un sitio estático. Simplemente abre el archivo `index.html` en tu navegador o súbelo a un servicio de hosting estático como GitHub Pages.

## Stack Tecnológico

- **HTML5**: Para la estructura de la aplicación.
- **CSS3**: Para los estilos y el diseño.
- **JavaScript (ES6+)**: Para toda la lógica de cálculo y la interactividad.
